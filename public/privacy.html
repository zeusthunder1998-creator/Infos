Privacy Policy

Last updated: April 21, 2026

Infos ("the app") is an admin portal used by a small team to manage internal notices, credentials, and game entries. This policy describes what data the app collects, how it's used, and how it's stored.

Who operates this app

Infos is operated by Thunder Anuprayog. For questions about this policy or your data, contact us at zeusthunder1998@gmail.com.

What data the app collects

  • Account credentials: username and password you set during sign-in. These are used solely to authenticate you within the app.
  • Content you create: notices, game entries, credential records, and any descriptions or links you enter. This content is shared only with other admins you've authorized.
  • Usage timestamps: when items were created or last updated, used to display "created X ago" indicators.
  • Local device preferences: your chosen theme (light/dark/auto) and last-active tab, stored in your device's local storage only. Never sent to our servers.

What the app does NOT collect

  • No advertising identifiers, device IDs, or tracking cookies.
  • No location data.
  • No contacts, photos, or files from your device.
  • No analytics or crash-reporting data sent to third parties.
  • No behavioral tracking of any kind.

How data is stored

Content and credentials are stored in a private Supabase database on Supabase-hosted servers. All network traffic between your device and the database uses HTTPS encryption.

QR code images uploaded by the main admin are stored in Supabase Storage as publicly-addressable files (anyone with the URL can view them, same as a hosted image).

Data sharing

We do not sell, rent, or share your data with third parties. Data stored in the app is visible only to authorized admins within your own instance.

Third-party services we rely on

  • Supabase (database hosting, realtime sync, file storage) — supabase.com/privacy
  • Vercel (web hosting) — vercel.com/legal/privacy-policy
  • Google Fonts (serves the Inter font) — policies.google.com/privacy

Data security

All communication between the app and the database is encrypted using HTTPS/TLS. The database itself is hosted by Supabase with standard production-grade security practices. Database access is restricted by API key configured at build time.

Data retention and deletion

Your data is retained for as long as your account exists. The main admin ("Zeus") can delete any sub-admin or co-admin account, and any content entry, at any time. When an account is deleted, its credentials are removed from the database. Content the user created may remain if it was also assigned to other admins.

To request full deletion of your data, contact zeusthunder1998@gmail.com.

Children's privacy

This app is not intended for use by children under 18. We do not knowingly collect data from children.

Permissions requested on Android

The Android version of Infos runs the web app in a trusted web view. It requests only the standard permissions needed to function: internet access and full-screen display. It does not request access to the camera, microphone, contacts, location, or any other sensor.

Changes to this policy

If this policy changes, the updated version will be posted at this URL. Major changes will be noted in the app's About Us screen.
